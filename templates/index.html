Web Development Course

The course assumes that students have already mastered basic python.

As part of the course, the basics of html / css are given sufficient for the formation of html pages in the browser, but not sufficient for professional layout.

The depth of mastering django will be sufficient for self-writing web projects that contain editable pages, structure of specific data, display from the front-end and create editing of this data.

All sections of django will be covered.
